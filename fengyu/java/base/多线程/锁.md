## java中的锁

#### java中的锁
在 Java 中主要2种加锁机制:
- synchronized关键字
- Lock接口
这两种机制的底层原理存在一定的差别

synchronized 关键字通过一对字节码指令 monitorenter/monitorexit 实现， 这对指令被 JVM 规范所描述。
java.util.concurrent.Lock 通过 Java 代码搭配sun.misc.Unsafe 中的本地调用实现的
